# Google Data Fetcher

A simple Node.js project that uses the SerpApi to fetch data from Google and saves the results to a CSV file. The project leverages SerpApi's search capabilities to gather search results based on the provided query.
- no duplicates domains
- no content from social media

## Prerequisites

Before you start, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (preferably the latest LTS version),
- An API key from [SerpApi](https://serpapi.com/). You must create a local .env file with your API_KEY

## Start
- `npm install`
- `npm run start`
