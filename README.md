# Google Data Fetcher

This project supports the paper "Web GEMs: Revealing Broken Access Control (BAC) Vulnerabilities in Web Applications Through Front-End Tampering". SerpApi is used to fetch data from Google and these are saved in a CSV file. The script excluded 1 duplicate domain.

## Setup

Before you start, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (preferably the latest LTS version),
- An API key from [SerpApi](https://serpapi.com/). You must create a local .env file with your API_KEY

## Start
- `npm install`
- `npm run start`
